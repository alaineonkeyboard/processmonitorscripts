You need to start from a fresh install of Ubuntu 14.04 64-bit for your server, and the default user with sudo/root access

# add to groups
$ sudo usermod -a -G sudo,www-data,adm,rvm <user> # may need to logout,login

$ sudo mkdir -p /var/www
$ sudo chown -R root:root /var/www

# the script needed to install all the requirements is in git, so we should install git first

$ sudo apt-get -y install git

$ sudo git clone https://github.com/atuaradil/processmonitorscripts.git /var/www/processmonitorscripts

$ cd /var/www

$ sudo processmonitorscripts/processmonitor-setup.sh

# you may need to logout and login again after the installation has finished, due to access
